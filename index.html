SJ Connect - Senior Junior Interaction Platform

SJ Connect is a front-end web application designed to enhance senior-junior interaction by providing a platform to share experiences, ask doubts, and build meaningful connections. The project features role-based access with user and admin logins, a DoubtBox for Q&A, and an Experience Wall where seniors can post insights for juniors to learn from. Built using HTML, CSS, and JavaScript (ES6) with LocalStorage for data persistence, SJ Connect offers a responsive, modern UI and smooth navigation across all sections without relying on a backend.
